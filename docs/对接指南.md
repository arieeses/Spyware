# 对接指南 · v2board + 1Panel/aaPanel

把真实面板接进内鬼系统,替掉 sample 演示数据。两条数据线:

```
1Panel/aaPanel ──(Nginx spyware 日志)──┐
                                        ├──▶ 内鬼系统
v2board MySQL ──(token/注册时间/流量)──┘
```

---

## 一、v2board(用户画像)

提供 token → 注册时间、流量、分组。**建只读账号**,别用 root。

### 1. 建只读库账号

```sql
CREATE USER 'spyware_ro'@'%' IDENTIFIED BY '强密码';
GRANT SELECT ON v2board.* TO 'spyware_ro'@'%';
FLUSH PRIVILEGES;
```

### 2. 填配置

```bash
cp config.example.json config.json
# 编辑 config.json, 填 host/port/user/password/database/prefix
```

> `prefix` 通常是 `v2_`(表名如 `v2_user`)。魔改版若改过表前缀,按实际填。

### 3. 同步

```bash
pip install pymysql          # 连接器唯一依赖
python3 -m spyware.cli sync-v2board
```

读 `v2_user` 的 `token / created_at(时间戳) / u+d(流量) / group_id / banned`,写入本地库。
可加进 cron 定时同步(如每 10 分钟)。

## 二、1Panel / aaPanel(订阅拉取日志)

### 1. 配置专用日志格式(面板里编辑站点 Nginx 配置)

```nginx
log_format spyware '$time_iso8601|$http_x_forwarded_for|$remote_addr|'
                  '$status|$request_time|$http_user_agent|'
                  '$arg_token|$request_uri';

location /api/v1/client/subscribe {
    access_log /www/wwwlogs/spyware_sub.log spyware;
    proxy_pass http://v2board_backend;
}
```

**CDN/反代必配**真实 IP 还原,否则 IP/ASN 信号失效:

```nginx
set_real_ip_from <你的CDN/反代网段>;
real_ip_header X-Forwarded-For;
real_ip_recursive on;
```

改完在面板里 reload Nginx。

### 2. 日志位置

| 面板 | 默认路径 |
|------|---------|
| aaPanel | `/www/wwwlogs/<域名>.log`(上面自定义的是 `spyware_sub.log`) |
| 1Panel | `/opt/1panel/apps/openresty/openresty/www/sites/<域名>/log/access.log` |

以面板「网站→日志」查看器显示的路径为准。

### 3. 导入(增量, cron 安全)

```bash
python3 -m spyware.cli ingest --log /www/wwwlogs/spyware_sub.log
```

`ingest` 记录每个文件的读取 offset,**重复运行只读新增行**;日志轮转(inode 变)或被截断会自动从头读。加进 cron:

```cron
*/5 * * * * cd /path/to/内鬼系统 && python3 -m spyware.cli ingest --log /www/wwwlogs/spyware_sub.log
```

> 中央面板与面板不在同一台机器时:在面板服务器上用 vector/fluent-bit/rsync 把 `spyware_sub.log` 推送/同步到中央机再 ingest,或直接在面板机跑 ingest 后把 `spyware.db` 汇聚。多机场纳管的完整方案见[架构设计.md](架构设计.md)增量 2/6。

## 三、查看结果

```bash
python3 -m spyware.cli analyze        # 命令行
python3 -m spyware.web                # 浏览器 http://127.0.0.1:8787
```

## 定时闭环(单机)

```cron
*/5  * * * * cd /path/to/内鬼系统 && python3 -m spyware.cli ingest --log /www/wwwlogs/spyware_sub.log
*/10 * * * * cd /path/to/内鬼系统 && python3 -m spyware.cli sync-v2board
```

Web 面板每次刷新即时重算,无需重启。
